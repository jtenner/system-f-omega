    Function showError

    • showError(err: TypingError): string

      Pretty-prints TypingError for user-facing diagnostics.

      Purpose: Human-readable errors with context (types/kinds shown). Covers all variants: unbound, mismatch, missing impl/case/field, cyclic, etc.

      import { freshState, inferType, varTerm, showError } from "system-f-omega";
