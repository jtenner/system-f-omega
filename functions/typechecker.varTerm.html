    Function varTerm

    • varTerm(name: string): { var: string }

      Constructs a term variable.

      Term variables refer to values bound in:

      • lambda abstractions (λx. body)
      • let-bindings (let x = value in body)
      • pattern matches (variables introduced by patterns)
      • the global or local typing context

      During inference, the variable’s type is looked up in the context using inferVarType.

      This is the term‑level analogue of varType for type variables.

      Parameters

      • name: string

        The variable name ("x", "f", "self", ...)

      Returns { var: string }

      A VarTerm node { var: name }

      Example

      varTerm("x");        // => "x"
      lamTerm("x", Int, varTerm("x"));
