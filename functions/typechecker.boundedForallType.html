    Function boundedForallType

    • boundedForallType(
          name: string,
          kind: Kind,
          constraints: TraitConstraint[],
          body: Type,
      ): Type

      Constructs a bounded universal type, also written:

      ∀name::kind where C₁, C₂, ... . body

      This represents trait‑bounded polymorphism: a type that is universally quantified over a type variable with additional constraints that the variable must satisfy.

      For example:

      ∀Self::* where Eq<Self>. (Self → Int)

      means: “For every type Self that implements Eq, the function has type Self → Int.”

      Components

      • name — the bound type variable ("Self", "A", "T", …)
      • kind — the kind of the variable (*, * → *, …)
      • constraints — list of trait requirements, each a TraitConstraint: { trait: "Eq", type: varType("Self") }
      • body — the type in which the bound variable is in scope

      How it is used

      Bounded foralls are produced by:

      • traitLamTerm (trait lambdas)
      • trait‑polymorphic functions and methods

      They are consumed by:

      • instantiateWithTraits
      • inferTraitAppType
      • autoInstantiate

      The idea:

      • A type variable introduced by a bounded forall must satisfy certain trait constraints.
      • When instantiating the type (e.g., during function application), the typechecker must find dictionaries that satisfy all constraints.

      Examples

      Single constraint

      boundedForallType(
        "Self",
        starKind,
        [{ trait: "Eq", type: varType("Self") }],
        arrowType(varType("Self"), conType("Int"))
      );

      // ∀Self::* where Eq<Self>. (Self → Int)

      Multiple constraints

      boundedForallType(
        "T",
        starKind,
        [
          { trait: "Eq",   type: varType("T") },
          { trait: "Show", type: varType("T") },
        ],
        varType("T")
      );

      Related

      • TraitConstraint — describes each constraint
      • instantiateWithTraits — performs instantiation and dictionary lookup
      • traitLamTerm — term‑level constructor for bounded polymorphism
      • checkType — checks trait lambdas against bounded forall types
