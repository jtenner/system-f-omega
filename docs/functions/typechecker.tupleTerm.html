    Function tupleTerm

    • tupleTerm(elements: Term[]): Term

      Constructs a tuple value:

      (e₁, e₂, ..., eₙ)

      Tuples are ordered, fixed‑size collections of terms. Unlike records, tuple fields are positional, not named.

      Examples of tuple terms:

      • (1, true)
      • (x, (y, z))
      • () — the empty tuple (unit)

      Type inference

      Handled by inferTupleType:

      • Infers the type of each element
      • Produces a corresponding TupleType

      For example:

      tupleTerm([1, true])  ⇒  (Int, Bool)

      Tuples also support indexing via tupleProjectTerm.

      Used in

      • constructing structured data
      • variant payloads (Cons(hd, tl))
      • recursive ADTs combined with muType
      • pattern matching (tuplePattern)

      Parameters

      • elements: Term[]

        The ordered list of element expressions

      Returns Term

      A TupleTerm node { tuple: [...] }

      Example

      tupleTerm([
        conTerm("1", conType("Int")),
        conTerm("true", conType("Bool"))
      ]);
      // => (1, true)

      Example: Unit tuple:

      tupleTerm([]);   // => ()
