    Function termBinding

    • termBinding(name: string, type: Type): { term: { name: string; type: Type } }

      Constructs a term binding for the typing context (Γ):

      name : type

      A termBinding represents a variable in scope with an associated type. These bindings are stored in the context and used during type inference whenever a variable (varTerm) is looked up.

      Examples of where term bindings are introduced:

      • lambda parameters (λx:τ. body adds x : τ)
      • let‑bindings (let x = value in body adds x : type(value))
      • global or top‑level definitions
      • dictionary variables introduced by trait lambdas

      inferVarType uses these bindings to determine the type of a variable term.

      Parameters

      • name: string

        The variable name being added to the context

      • type: Type

        The type assigned to that variable

      Returns { term: { name: string; type: Type } }

      A TermBinding node suitable for insertion into state.ctx

      Example

      termBinding("x", conType("Int"));
      // => { term: { name: "x", type: Int } }
